## 🎬 The 4 Steps (Summary)

1. **Create GitHub repo** (3 min)
   - Go to github.com/new
   - Name it `dyf-website`
   - Make it Public
   - Copy the URL

2. **Push code** (2 min)
   ```bash
   cd dyf-website
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin YOUR_GITHUB_URL
   git push -u origin main
   ```

3. **Create Cloudflare account** (3 min)
   - Go to pages.cloudflare.com
   - Sign up (free)

4. **Deploy** (5 min)
   - Connect GitHub
   - Select repository
   - Click "Deploy"
   - Done!
